# TaskFlow — SaaS Landing Page

A modern, responsive landing page for a fictional project management SaaS, built with Next.js 16, TypeScript, and Tailwind CSS.

**Live demo:** https://taskflow-landing-ten.vercel.app/

## Features

- Fully responsive design (mobile, tablet, desktop)
- Dark mode support
- Interactive FAQ accordion with React state
- Working contact form with Formspree integration
- Clean, data-driven component architecture
- Optimized for Core Web Vitals

## Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Forms:** Formspree
- **Deployment:** Vercel

## Run locally

\`\`\`bash
git clone https://github.com/Pacmyan/taskflow-landing.git
cd taskflow-landing
npm install
npm run dev
\`\`\`

Open http://localhost:3000
